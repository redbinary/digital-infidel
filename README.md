#### Digital Infidel ####
https://github.com/redbinary/digital-infidel/

Simple AsteroidOS watchface.

I made this for my Asus ZenWatch 2 'Sparrow' which has a square face. Not sure how it will layout for anything else. watches such as the Asus ZenWatch running AsteroidOS.

Time and date are centered.

Includes 4 widgets, one in each corner. These can be disabled so as not to display or use resources by using the corresponding useWidget_xxx flag. (I added this when I found out my Sparrow didn't have step count support yet.)

Widgets included are weather, battery, notifications, and steps.

I borrowed the mass of the step counter from https://github.com/AsteroidOS/unofficial-watchfaces/tree/master/digital-weather-hrm-steps
hopefully I have attributed appropriately.

I just made this watchface for myself but would feel remiss not making it available regardless of the quality.
